# Re:merge — StyleHub B2B

AWS 기반 패션 B2B 무역구매·배송대행 플랫폼

- 일반 구매: 상품 조회 → 장바구니 → 주문 → 결제 → 배송 → 거래 확정
- 소싱 구매: 소싱 요청 → 견적 작성 및 협의 → 견적 채택 → 주문

전체 B2B 거래 프로세스를 온라인으로 처리하는 서비스입니다.

## 프로젝트 개요

| 항목    | 내용                                              |
|-------|-------------------------------------------------|
| 프로젝트명 | AWS 기반 B2B 무역구매/배송대행 서비스 (StyleHub B2B)         |
| 개발 기간 | 2026.06.02 ~ 2026.07.09 (약 5.5주)                |
| 팀 구성  | 5인 (백엔드/프론트엔드 풀스택 병행)                           |
| 산출물   | 웹 서비스(백엔드 API + 프론트엔드 SPA), ERD, 요구사항정의서, 발표 자료 |

패션 도매/무역 거래에서 반복되는 소싱 요청 → 견적 대조 → 협상 → 계약 → 발주 → 정산 흐름을 하나의 플랫폼으로 통합해, 바이어와 셀러 양측이 거래 이력을 추적 가능한 상태로 관리할 수 있도록 설계했습니다.

## 기술 스택

### Backend

| 구분         | 기술                                         |
|------------|--------------------------------------------|
| 언어 / 프레임워크 | Java 17, Spring Boot 3.5.15                |
| 인증/보안      | Spring Security, JWT (jjwt), OAuth2 Client |
| 데이터 / 캐시   | Spring Data JPA, MySQL, Spring Data Redis  |
| 실시간 / 비동기  | Spring WebSocket (STOMP), Spring WebFlux   |
| 매핑 / 문서 생성 | MapStruct, iText (전자계약 PDF)                |
| 알림 발송      | Spring Mail, CoolSMS SDK                   |
| 기타         | Spring Boot Actuator, Lombok               |

### Frontend

| 구분           | 기술                                                |
|--------------|---------------------------------------------------|
| 프레임워크 / 빌드   | React 18.3.1, Vite 6                              |
| 스타일링         | Tailwind CSS 4, MUI, Radix UI                     |
| 상태 / 라우팅 / 폼 | Zustand, React Router 7, React Hook Form          |
| 통신           | Axios, @stomp/stompjs (WebSocket)                 |
| 결제 / 전자서명    | TossPayments SDK, PortOne SDK, signature_pad      |
| UX 보조        | Recharts, react-hot-toast, sonner, embla-carousel |

## 핵심 기능

| 도메인      | 주요 기능                                                           |
|----------|-----------------------------------------------------------------|
| 인증/인가    | JWT + Spring Security 인증, Redis 기반 리프레시 토큰, 아이디/비밀번호 찾기, 자동 로그인 |
| 회원가입     | 사업자등록증 OCR 인식 + 국세청 진위 확인 연동                                    |
| 회사/직원 관리 | 회사 정보 관리, 직원 초대/권한 관리, 역할별(ADMIN/PRESIDENT/EMPLOYEE) 대시보드       |
| 상품       | 카테고리 기반 상품 CRUD, 다중 이미지/PDF 업로드, 옵션(SKU) 관리, 위시리스트              |
| 소싱       | 소싱 요청 등록, 공급사 자동/수동 배정, 관리자 승인·반려                               |
| 견적       | 소싱 요청 기반 견적서 생성/제출, 상태 관리(제출/열람/승인/반려)                          |
| 협의       | 셀러-바이어 간 조건 협상, 재견적/재계약 버전 체이닝(V-Chaining)                      |
| 전자계약     | 손서명 기반 전자서명, 계약서 PDF 자동 생성 및 해시 검증                              |
| 장바구니/결제  | 장바구니, Checkout, 토스페이먼츠 연동, 무통장입금 지원                             |
| 주문       | 주문 상태 관리, 출고 처리, 거래 확정, 배송 추적                                   |
| 정산       | 플랫폼 수수료 계산, 정산 대시보드                                             |
| 이의제기     | 주문 건별 클레임 접수 및 증빙 파일 첨부, 처리 워크플로                                |
| 알림       | Redis + SSE 기반 실시간 알림                                           |
| 1:1 문의   | STOMP + WebSocket 기반 실시간 채팅                                     |
